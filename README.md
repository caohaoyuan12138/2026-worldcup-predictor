# ⚽ 2026世界杯预测系统 v3.0

融合 Elo 等级分 + 泊松分布(Dixon-Coles) + 经济学模型 + 赔率市场 的四维足球预测引擎，并集成 AI 推演裁判进行深度研判。

## 🏗️ 系统架构

```
前端 (SPA) ←HTTP→ Node.js 后端 (server.mjs)
                        ├─ 数学模型 (engine.mjs v3.2)
                        │   ├─ Elo等级分 (48队真实数据)
                        │   ├─ 泊松+Dixon-Coles (ρ动态0.01-0.05)
                        │   ├─ 经济学模型 (48国GDP/人口/气候)
                        │   └─ 赔率市场模型 (含让球盘口)
                        │   └─ 战意修正 (出线形势→末轮λ调整)
                        ├─ AI推理裁判 (reasoning_agent.py)
                        │   └─ deepseek-v4-flash (推演裁判)
                        └─ 数据库 (worldcup.json v5)
                            ├─ 48队/72组历史交锋/479条近10场
                            ├─ 48场赔率数据(含让球盘口)
                            ├─ 小组出线形势 (12组)
                            └─ 淘汰赛对阵树 (1/16→决赛 26场)
```

## 🚀 快速启动

```bash
# 安装依赖（Node.js ≥ 18）
npm install

# 启动服务器
node server.mjs

# 浏览器打开

```

## 🧠 模型说明

### 四维融合权重（可配置）
| 模型 | 权重 | 说明 |
|------|------|------|
| Elo等级分 | 25% | FIFA排名初始化，按已赛结果更新 |
| 泊松分布 | 30% | 含Dixon-Coles低比分修正+近10场+历史交锋 |
| 经济学 | 10% | 48国GDP/人口/气候因子 |
| 赔率市场 | 35% | 手动输入赔率/让球盘口/隐含赔率 |

### AI 推演裁判
使用 deepseek-v4-flash 大模型，接收数学模型输出的因子向量，进行推理链分析，输出结构化研判报告。支持赔率、让球盘口、战意等因子。

### 特殊功能
- **让球盘口联动**：盘口深度影响λ，让-2可使主队λ提升40%+
- **战意修正**：末轮出线形势影响λ（需赢球×1.12，已出线×0.90）
- **动态DC ρ**：实力接近ρ=0.05(增平局)，悬殊ρ=0.01
- **出线形势**：12组实时积分+淘汰赛对阵树
- **复盘分析**：48场已完赛回测，偏差计算+改进建议
